Embedded Java Webcontainer

Ist das nicht cool? Dieses HTML wird von einem Web Container ausgeliefert, der in eine Java-Anwendung eingebettet wurde.

Mit Jetty als Embedded Web Container ist so einiges möglich, aber es kann natürlich schnell kompliziert werden. Darum haben wir uns hier auf eine einfache Lösung mit statischen HTML-Dateien und Servlets beschränkt. JSP, JAX-WS könnte man sicher auch irgendwie zum Laufen bringen, das ist mir aber zu kompliziert. :-)

Ein einfaches Servlet geht aber: HelloServlet

Und hier ein Servlet, das ein Template zur HTML-Erzeugung nutzt: TemplateServlet
