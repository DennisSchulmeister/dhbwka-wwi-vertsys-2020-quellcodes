Dieses Projekt wurde mit Spring Boot erstellt und nutzt Maven zum Compilieren
und Bauen der Auslieferungspakete. Es kann mit jeder IDE bearbeitet werden, die
Maven-Projekte unterstützt.

Auf der Kommandozeile kann das Projekt mit folgendem Befehl gestartet werden:

    mvnw spring-boot:run

Falls dies nicht funktioniert:

    mvn spring-boot:run

Auf diese Weise werden die benötigten Bibliotheken heruntergeladen, der
Quellcode compiliert und das Projekt gestartet.
